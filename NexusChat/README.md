# Nexus Chat Messenger

https://youtu.be/1OEEjahPX18?si=BPr1PmO-Bh7TFjKp

📌 Motivation

As students at the University of Windsor and strong advocates for digital privacy, we are driven to address growing concerns around online data security within our community. Platforms like Discord, widely used among Computer Science students, are increasingly introducing requirements such as Face ID and additional personal data collection, raising valid concerns about user privacy.

This issue lies at the intersection of ethics, technology, and user autonomy, highlighting the ongoing tension between convenience and personal data protection. Nexus was created to challenge this trend by offering a privacy-first alternative.

Building Nexus presents a significant technical challenge, requiring modern frameworks and scalable architecture to deliver a secure, efficient, and user-friendly platform. While inspired by our university environment, the solution is designed to benefit users globally by promoting digital ethics and giving individuals greater control over their personal data.

🚀 Project Overview

Nexus is a user-focused chat platform that delivers advanced messaging and customization features—without relying on intrusive big-tech practices.

The system is developed through a structured, scalable process, progressing from a simple terminal-based chat application to a fully featured GUI messenger with a robust backend and database.

🛠️ Tech Stack & Architecture

Backend: Spring Boot

Frontend / GUI: Java Swing

Database: MySQL, SQLite

ORM: Hibernate, JPA
